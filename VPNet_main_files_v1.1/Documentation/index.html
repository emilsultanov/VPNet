INTRODUCTION

VPNet is a Multipurpose VPN and Cloud Service Template with WHMCS designed for all kinds of VPN, Cloud Service, VOIP, Saas, Technology, Domain and Hosting Business. It's an HTML5 template based on latest Bootstrap v4.0.0. It includes WHMCS Template based on WHMCS v7.5.0. Anyone can easily update/edit this plugin by following our Well Sorted Documentation.

Mainly, our template is an HTML template and we just provided support by customizing WHMCS header, footer and color.

Main Features of "VPNet":

  • - Latest Stable Bootstrap v4.0.0 Ready
  • - Fully Responsive
  • - Retina Ready
  • - 10 Default Color with Color Switcher (Color switcher is only for demo).
  • - Clean Design
  • - Google Font
  • - Latest FontAwesome Icons (v5.0.8)
  • - Modern Cross Browser Support
  • - W3C Validated Code
  • - WHMCS v7.5.0 Ready
  • - Working Contact Forms (Validated by parsley JS)

HTML STRUCTURE

The "VPNet" template is a responsive HTML template based on Bootstrap v4.0.0. The structure is based on a flexible grid that lets you change HTML pages quickly and logically with a nestable system.

If you would like to edit the layout of any elements in the whole template, you would do the following:

  • 1. Open any '.html' or '.css' file in a text editor.
  • 2. Find related comment/code.
  • 3. Do your magic.

INSTALLATION

Via FTP:

1. Open up your 'FTP Software' of choice and connect to your hosting

2. Browse to required directory (Usually: ./public_html)

3. Unzip the 'main_files.zip' and choose templates to upload.

4. Upload the template to the server.

FAQ

Once you finish setup successfully, you can customize it as you want. Some common frequently asked questions are given bellow.

Q. How to update existing theme?

  • Make sure to backup all your previous files.
  • Check 'Change log' to know the new changes.
  • The best way to update is to use a compare tool, like: Code Compare. And use it to compare the 'folders/files' (the version you're using with the new version).
  • ***Updating HTML Templates Requires Manual Work.
  • Replace 'custom.css' file or any other content from your backup.

Q. How to change default color schemes?

  • Open '.html' file in a text editor. (For example: 'index.html') or open 'includes/head.tpl' from WHMCS template.
  • Find the file 'theme-color-1.css'
  • Change the file name. (10 different color themes are provided in 'css/colors' folder)

Q. How to remove preloader?

  • Open '.html' file in a text editor (For example: 'index.html').
  • Find the comment 'Preloader'
  • Remove code block.

Q. How to add image to support retina display?

  • Make 2 copies of your image. One default image (Example: file.png) and another retina supported image (Example: file@2x.png).

Q. How to change logo?

  • Open '.html' file in a text editor (For example: 'index.html').
  • Find the comment 'Logo'
  • Edit image name.

Q. How to change header social icons and links?

  • Open '.html' file in a text editor (For example: 'index.html').
  • Find the comment 'Header social icons'
  • Edit/replace code.

Q. How to change Header menu?

  • Open '.html' file in a text editor (For example: 'index.html').
  • Find the comment 'Header-menu'
  • Edit/replace code.

Q. How to change Banner content?

For main title and buttons:
  • Open '.html' file in a text editor (For example: 'index.html').
  • Find the comment 'Banner content'
  • Edit/replace code.
For animated subtitle:
  • From js folder open 'scripts.js' file in a text editor.
  • Find the comment 'Typed JS'
  • Edit/replace texts.

Q. How to change Pricing plans?

  • Open '.html' file in a text editor (For example: 'index.html').
  • Find the comment 'Pricing slider'
  • Edit/replace code.

Q. How to change Reviews?

  • Open '.html' file in a text editor (For example: 'index.html').
  • Find the comment 'Reviews'
  • Edit/replace code.

Q. How to change Footer info?

  • Open '.html' file in a text editor (For example: 'index.html').
  • Find the comment 'Footer info'
  • Edit/replace code.

Q. How to change footer newsletter from action?

  • Open '.html' file in a text editor (For example: 'index.html').
  • Find the comment 'Footer newsletter'
  • Edit/replace code.

Q. How to change copyright?

  • Open '.html' file in a text editor (For example: 'index.html').
  • Find the comment 'Copyright'
  • Edit/replace code.

Q. How to change contact form submission email?

  • Open 'sendmail.php' file in a text editor.
  • Find the comment 'Replace this with your own email address'
  • Edit email and name with your own.
